Soundbox: An Interactive Instrument | Fall 2015:

Soundbox is an interactive instrument that allows users to create their own tunes by selecting which notes are played, or not, from premade rotating discs. The user interacts with Soundbox via sensors.

Skills and Lessons learned:

  • Programming with Arduino.
  • Basic electronic wiring.
